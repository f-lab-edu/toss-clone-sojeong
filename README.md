# vanilla js 로 typescript 이용해서 SPA 구현하기


### 기술 스택
- typescript
- npm
- webpack
- mws(mock data 라이브러리)
  

### 프로젝트 구성
- class형 컴포넌트 구성
- 라이프사이클 관리를 위한 core Component class를 상속해서 구조화
- history API와 CustomEvent를 이용한 라우팅 처리
